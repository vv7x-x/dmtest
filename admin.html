إعدادات الإشعارات البريدية

هذه الإعدادات تُمكّن إرسال إيميلات فورية للأدمن عند وصول طلب جديد أو رسالة تواصل — حتى لو كان الموقع مقفول.

من Supabase Dashboard → Settings → API → anon public key

كيف تعمل؟

1) تأكد من تشغيل pg_net extension في Supabase Dashboard → Database → Extensions

2) أنشئ Edge Function: supabase functions deploy send-admin-notification --no-verify-jwt

3) أضف المتغيرات السرية: supabase secrets set RESEND_API_KEY=re_xxx ADMIN_EMAIL=admin@example.com STORE_URL=https://yoursite.com

4) احفظ الإعدادات أعلاه — وستصلك الإشعارات فور وصول أي طلب أو رسالة.
